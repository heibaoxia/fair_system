# Fair-System 智能项目分配与工作流编排系统

**Fair-System** 是一个致力于解决团队内部协作中“任务分配不均”难题的智能工作流管理平台。它不仅提供可视化的依赖关系流连线画板，还可以根据**工作量、难度、枯燥程度、工作强度**四个维度的多重评估权重，智能计算出最公平的模块分配方案和业绩蛋糕切分逻辑。

## 🌟 核心功能亮点

1. **可视化模块蓝图**：抛弃干巴巴的任务列表，使用直观的连线画板（Node Graph）动态设计任务的先后置依赖关系，通过数学射线算法实现了丝滑的卡片拖拽与连线选中高亮功能。
2. **多维公平分配引擎**：集成基于规则评分的统筹分配算法，前端调节权重滑杆（难度/工时等），后端计算分配方案，并在完成时根据打分加权计算每个人的奖金分成。
3. **“先锋服”设计原则**：极简流畅的 UI 交互，所有节点数据和依赖关系在客户端一键打包存储到后端。

## 🛠️ 技术栈

*   **后端**: Python + FastAPI
*   **数据库**: SQLite + SQLAlchemy (异步配置结构)
*   **前端交互**: 原生 JavaScript + 动态 SVG 渲染 + 纯鼠标坐标系连线数学碰撞检测
*   **UI/样式**: 纯原生 CSS，不依赖沉重的第三方库，极简轻量。

## 🚀 快速启动指北

### 1. 环境准备

确保你已经安装了 Python 3.10+ 环境。打开终端，在项目根目录下创建一个虚拟环境并激活：

```bash
# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境 (Windows)
.venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

### 2. 数据库初始化与假数据注入

初次运行项目之前，我们需要建表并灌入一些演示用的假数据，方便你在前端就能直接看到效果。

```bash
# 初始化创建数据库表
python init_db.py

# 灌入测试演示数据
python seed_demo_data.py
```

### 3. 运行服务

启动 FastAPI 内置的 Uvicorn 服务器：

```bash

`uvicorn app.main:app --reload``

随后打开浏览器访问：[http://127.0.0.1:8000](http://127.0.0.1:8000) 即可看到精美的仪表盘！
